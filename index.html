🔥 Firebase Real-Time Setup

  1. Go to console.firebase.google.com → Add project → name it anything → Continue
  2. Left sidebar → Build → Realtime Database → Create database → pick region → Start in test mode → Enable
  3. Click ⚙️ Project Settings → General tab → scroll to Your apps → click </> Web icon → register app → copy the firebaseConfig values
  4. Paste each value into the fields below and click Save & Connect
  5. Push this HTML file to GitHub → repo Settings → Pages → Source: main / (root) → your URL goes live!

Set these Realtime Database Rules (Database → Rules tab):

{"rules":{".read":true,".write":true}}
